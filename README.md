# Rebrith - Real-Time Battle

A retro-style real-time battle game featuring a Spellsword vs. Mage duel.

## How to Play

Defeat the enemy bot by managing your energy and using your skills effectively.

### Controls

**Desktop (Keyboard):**
- **A / D**: Move Left / Right
- **Q**: Attack (Costs Energy)
- **W**: Block (Mitigates Damage)
- **E**: Skill (Special Ability)
- **R**: Charge (Regenerate Energy)

- **Left Joystick**: Tap Left/Right arrows to move.
